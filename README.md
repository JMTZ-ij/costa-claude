# Costanera AI Sales Studio

Claude Code skills suite for Costanera Ltd's prospecting workflow. Paste a URL, get a scored operational audit and a ready-to-send three-tier retainer proposal — in Costanera's voice, scoped to a 2-person team.

This version is rewritten end-to-end for Costanera's specific positioning: Airtable + automation + AI integration as a monthly retainer.

---

## What it does

Type one command in Claude Code:

```
> /costa prospect https://acmepools.com

Launching 5 parallel agents...
✓ Operational Pain (30%)        — Score: 78/100
✓ Data Stack Fit (20%)          — Score: 72/100
✓ Automation Surface (20%)      — Score: 81/100
✓ AI Fit (10%)                  — Score: 65/100
✓ Decision Maker Access (20%)   — Score: 76/100

Costanera Fit Score: 75/100 (Grade: A — Strong fit)
Recommended Tier: Operate @ $7,500/mo

Top 3 Pain Points:
  1. Sales Ops Coordinator role posted last week — manual data entry
  2. Multiple inboxes, no triage layer, 24-hour response SLA
  3. Shopify and QuickBooks not connected (manual reconciliation)

Estimated Hours Saved/Month: 47 hrs
Estimated $/Month Saved:     $3,290

Full report saved to: COSTANERA-PROSPECT.md
```

Then `/costa outreach <prospect>` drafts the cold email in Eduardo's voice, and `/costa proposal <client>` expands into a full three-tier proposal.

## Install

```bash
curl -fsSL https://raw.githubusercontent.com/<your-org>/costa-claude/main/install.sh | bash
```

Or from a local clone:

```bash
git clone https://github.com/<your-org>/costa-claude.git
cd costa-claude
./install.sh
```

The installer copies skills to `~/.claude/skills/` and agents to `~/.claude/agents/`. Re-run anytime to pick up updates.

To install from a different repo, set `COSTANERA_REPO`:

```bash
COSTANERA_REPO=your-username/costa-claude curl -fsSL ... | bash
```

## Commands

| Command | What it does | Output |
|---------|-------------|--------|
| `/costa prospect <url>` | Full analysis + tier proposal | `COSTANERA-PROSPECT.md` |
| `/costa quick <url>` | 60-second pass/maybe/skip check | Terminal only |
| `/costa audit <url>` | Operational audit (no proposal) | `COSTANERA-AUDIT.md` |
| `/costa proposal <client>` | Three-tier retainer proposal | `COSTANERA-PROPOSAL.md` |
| `/costa outreach <prospect>` | Cold email — 3 variants | `COSTANERA-OUTREACH.md` |
| `/costa icp [description]` | Define/view your ICP | `COSTANERA-ICP.md` |

Recommended flow:

```
1. /costa icp                 # one time — lock in your ICP
2. /costa quick <url>         # 60-second gut check
3. /costa prospect <url>      # full analysis + proposal (if quick passed)
4. /costa outreach <prospect> # cold email in Costanera voice
```

## The three tiers

Every proposal generated by this suite offers these:

| Tier | Price (USD/mo) | For |
|------|---------------|-----|
| **Foundation** | $3,000–$4,500 | 1–15 ppl, single pain, 3-month minimum |
| **Operate ★** | $6,000–$8,500 | 15–50 ppl, multi-system, 6-month minimum |
| **Transform** | $11,000–$15,000 | 50+ ppl, custom software layer, 6-month minimum |

★ = recommended default. Most engagements should land here.

One-time onboarding fee: 50% of one month's retainer. 10% discount on annual prepay. Prices and tier definitions live in `costa/SKILL.md` — edit there if you want to adjust.

## Scoring methodology

`Costanera Fit Score = weighted average of:`

| Category | Weight | Measures |
|----------|--------|----------|
| Operational Pain | 30% | Manual processes, broken handoffs, spreadsheet hell |
| Data Stack Fit | 20% | Airtable applicability, integration debt |
| Automation Surface | 20% | Number/value of automatable workflows |
| AI Fit | 10% | Productive (not gimmicky) AI use cases |
| Decision Maker Access | 20% | Reachable founder/ops buyer at right size |

Score ranges:
- **85–100 (A+)** — send outreach within 48 hours
- **70–84 (A)** — outreach this week
- **55–69 (B)** — outreach when capacity allows
- **40–54 (C)** — skip unless warm intro
- **0–39 (D)** — disqualify

The 10% AI weight is deliberately small. Costanera's blog post "[You Don't Need AI. You Need Software.](https://www.costanera.uk/blog/you-dont-need-ai-you-need-software/)" is the philosophy: automation and software fix most ops problems; AI is the icing.

## Voice

All outputs match Costanera's site voice:
- Plain English, anti-jargon
- Founder-to-founder
- Hours and dollars, not adjectives
- Honest about limits (a 2-person team can't do everything)
- Sign-off as Eduardo Casanova, Founder

The voice rules live in `costa/SKILL.md` and propagate through every sub-skill.

## Customization

Edit any of these to tune the suite:

- `costa/SKILL.md` — tier pricing, voice rules, business types served
- `skills/costa-icp/SKILL.md` — default ICP that pre-populates `COSTANERA-ICP.md`
- `agents/costa-*.md` — agent scoring weights and analysis logic
- `scripts/analyze_prospect.py` — `TOOL_SIGNALS`, `PAIN_PHRASES`, `BUSINESS_TYPE_HINTS`

Re-run `./install.sh` after edits.

## Uninstall

```bash
./uninstall.sh
```

Or remotely:

```bash
curl -fsSL https://raw.githubusercontent.com/<your-org>/costa-claude/main/uninstall.sh | bash
```

Removes all `costa*` skills, agents, and scripts from `~/.claude/`. Generated `COSTANERA-*.md` files in your working directories are left alone.
